What is IIFE ?

Posted on 2018-04-14

IIFE (Immediately Invoked Function Expression) is a JavaScript function that runs as soon as it is defined.

It is a design pattern which is also known as Self-Executing Anonymous Function and contains two major parts.

The first is the anonymous function with lexical scope enclosed within the Grouping Operator ( ). This turns the code within (function(){...}) into an expression.

The second part is creating the immediately executing function expression (), through which the JavaScript engine will directly interpret the function.

Thus, IIFE will be express like (function(){...})(). This pattern prevents accessing variables within the IIFE idiom as well as polluting the global scope.

Example

The function becomes a function expression which is immediately executed. The variable within the expression can not be accessed from outside.

Assigning the IIFE to a variable does not store it but its result.
