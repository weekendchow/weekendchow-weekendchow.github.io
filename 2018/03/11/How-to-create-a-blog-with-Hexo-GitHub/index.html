How to create a blog with Hexo + GitHub

Posted on 2018-03-11

There are many ways to build a static blog and I choose Hexo + GitHub pages. We can use Hexo to write blogs locally. Hexo translates the blog into a static page and then uploads these static pages to github. These static pages are deployed by the static web server of github pages.

What is Hexo?

Hexo is a fast, simple & powerful blog framework based on Node.js. We write posts in Markdown and Hexo generates static files as well as deploy them in seconds.
